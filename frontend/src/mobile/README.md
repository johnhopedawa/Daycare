# Deprecated Mobile-Web Reference

`frontend/src/mobile/` is reference-only.

This code was built for the abandoned Capacitor/mobile-web path and is no longer part of the runtime architecture.

Use it only for:
- screen hierarchy reference
- naming reference
- rough UX flow comparison

Do not:
- import this folder into the main web app again
- use it as the production mobile runtime
- treat it as React Native code

The real mobile app now lives in [`native-mobile/`](../../../native-mobile/README.md).
